Команда кросс-функциональная

Но её члены часто имеют ключевую компетенцию

Сколько весит элемент бэклога

Если его реализация требует нескольких членов команды

  1. Сумма оценок составляющих элементов.
  2. Максимальная из оценок составляющих элементов.
  3. Средняя из оценок составляющих элементов.

Важно держать в уме

  1. Над одним элементом бэклога могут работать несколько членов команды.
  2. Не всегда члены команды могут полноценно заменять друг друга.
  3. У каждого члена команды своя ёмкость.

F = 6us1 + 5us2 + 4us3 +3us4 +2us5 + us6 → max

3us1 + 5us2 + 3us3 +3us4 + 5us5 + 2us6 ≤ 15

3us1 + 5us2 + 3us3 +3us4 + 3us6 ≤ 15

5us1 + 13us2 + 5us3 +8us4 ≤ 15

5us1 + 8us2 + 5us3 + 5us4 ≤ 15

Распределение загрузки членов команды (SA, JS/ TS, Java, QA)
usi ≥ 0, usi ≤ 1, usi - целое, i = 1, …, 6
